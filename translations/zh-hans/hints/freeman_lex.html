在 Python 中，字符串可以解释为序列。这意味着，您可以使用for 循环遍历给定的字符串。 

for char in line:

如何处理循环中的每个字符？

实际上有两种情况：如果字符与前一个字符不相同（已经添加到结果字符串中），或者它们相等。如果结果字符串为空，则与不同字符的情况相同。使用if/else 语句。

那么，让我们来看看第一种情况。如果结果字符串为空，或者当前字符和之前字符不同： 

if not res or char!= res[-1]:
    ...

在这种情况下，您只需将当前字符添加到结果字符串中。 

if not res or char!= res[-1]:
    res += char
另一种情况是什么？

根据描述，如果相邻的两个字母相同，您需要删除它们。其中一个是当前字符，第二个已经添加到结果字符串中。在这种情况下，您只需要使用string slicing 删除结果字符串中的最后一个字符。 

else:
    res = res[:-1]
那么，我的最终解决方案应该是怎样的呢？

您只需将之前所有的提示连接起来，并将代码封装为函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def adjacent_letters(line: str) -> str:

    res = ""
    for char in line:
        if not res or char!= res[-1]:
            res += char
        else:
            res = res[:-1]

    return res

使用正则表达式模块和递归找到两个相同的相邻字母，用空行替换它们，并使用新字符串作为新函数调用的参数。如果没有相同的相邻字母，则返回空行。 

from re import sub, search


reg = r'(.)\1'

def adjacent_letters(line: str) -> str:

    return line if not search(reg, line) else adjacent_letters(sub(reg, '', line))
