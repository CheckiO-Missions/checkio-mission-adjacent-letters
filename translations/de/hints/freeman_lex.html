Fangen Sie eine Idee 💡! Iterieren Sie über die gegebene Zeichenkette und fügen Sie jedes Zeichen zu einer resultierenden Zeichenkette hinzu, wenn es nicht mit dem vorherigen Zeichen identisch ist. Andernfalls entfernen Sie einfach das letzte Zeichen aus der resultierenden Zeichenfolge.

Ok, womit soll ich anfangen?

Lassen Sie uns eine resultierende Zeichenkette deklarieren, die Sie am Ende zurückgeben. 

res = ""
...
return res

Eine Zeichenkette kann in Python als eine Sequenz interpretiert werden. Das bedeutet, dass Sie mit der Schleife for über die angegebene Zeichenkette iterieren können. 

for char in line:
    ...
Was tun Sie mit jedem Zeichen innerhalb der Schleife?

Es gibt eigentlich zwei Fälle: wenn das Zeichen nicht dasselbe ist wie das vorherige (bereits zur resultierenden Zeichenkette hinzugefügt) oder wenn sie gleich sind. Wenn die resultierende Zeichenkette leer ist, handelt es sich um denselben Fall wie bei unterschiedlichen Zeichen. Verwenden Sie die Anweisung if/else.

Schauen wir uns also den ersten Fall genauer an. Wenn die resultierende Zeichenfolge leer ist oder die aktuellen und vorherigen Zeichen unterschiedlich sind: 

if not res or char!= res[-1]:
    ...

In diesem Fall fügen Sie einfach das aktuelle Zeichen zur resultierenden Zeichenfolge hinzu. 

if not res or char!= res[-1]:
    res += char
Was sollte der andere Fall sein?

Gemäß der Beschreibung müssen Sie beide benachbarten Buchstaben entfernen, wenn sie gleich sind. Einer der beiden Buchstaben ist ein aktuelles Zeichen, der zweite ist bereits in der resultierenden Zeichenfolge enthalten. In diesem Fall müssen Sie nur das letzte Zeichen mit string slicing aus der resultierenden Zeichenfolge entfernen. 

else:
    res = res[:-1]
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def adjacent_letters(line: str) -> str:

    res = ""
    for char in line:
        if not res or char!= res[-1]:
            res += char
        else:
            res = res[:-1]

    return res

Finden Sie mit dem Modul für reguläre Ausdrücke und Rekursion zwei gleiche benachbarte Buchstaben, ersetzen Sie sie durch eine leere Zeile und verwenden Sie diese neue Zeichenkette als Argument für einen neuen Funktionsaufruf. Geben Sie die Zeile zurück, wenn es keine gleichen benachbarten Buchstaben mehr gibt. 

from re import sub, search


reg = r'(.)\1'

def adjacent_letters(line: str) -> str:

    return line if not search(reg, line) else adjacent_letters(sub(reg, '', line))
