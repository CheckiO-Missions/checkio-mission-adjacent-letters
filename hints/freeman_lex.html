A string may be interpreted as a sequence in Python. It means, you can iterate over the given string using for loop.

What to do with each character inside loop?

There are actually two cases: if the character is not the same as previous (already added to the resulted string) or they are equal. If the resulted string is empty, it's the same case as different characters. Use if/else statement.

So, let's look closer to the first case. If your resulted string is empty or current and previous characters are different: 

if not res or char != res[-1]:
    ...

In this case you just add current character to the resulted string. 

if not res or char != res[-1]:
    res += char
What should be the other case?

According to the description, you need to remove both adjacent letters if they are the same. One of them is a current character, the second is already added to the resulted string. In this case you just need to remove the last character from the resulted string using string slicing. 

else:
    res = res[:-1]
So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def adjacent_letters(line: str) -> str:

    res = ""
    for char in line:
        if not res or char != res[-1]:
            res += char
        else:
            res = res[:-1]

    return res

Using regular expression module and recursion find two same adjacent letters, replace them with empty line and use this new string as argument for a new function call. Return the line if there no equal adjacent letters left. 

from re import sub, search


reg = r'(.)\1'

def adjacent_letters(line: str) -> str:

    return line if not search(reg, line) else adjacent_letters(sub(reg, '', line))
